หัวข้อที่เรียนวันนี้ เรื่องของ array 
ึคุณสมบัติพื้นฐานของ array เป็นแบบ pointer หรือการชี้ไปที่ object ไม่ใช่ตัวของมันเอง 
ตัวแปร เหมือนกับกล่องที่แปะคุณสมบัติของมันเอง แต่ type ของ object จะเป็นป้ายบอกทางในการไปหากล่องของมัน เพราะงั้นต่อให้เราตั้งตัวแปรเป็น const ก็สามารถเปลี่ยนแปลงคุณสมบัติของมันได้ เพราะconst ส่งผลกับป้ายไม่ใช่กล่องคุณสมบัติ
array คือ set ของข้อมูลที่อยู่ใน bracket จะเป็นtype อะไรก็ได้ มีสมบัติของindex และการเรียงลำดับ
เราสามารถใช้ .length เพื่อบอกจำนวนของ dataข้างใน
ใช้[ตัวเลข] เพื่อระบุตำแหน่งของข้อมูล
.push เพื่อนำข้อมูลไปต่อท้าย ได้ทุกtype ของข้อมูล 
.pop กับ .shift เป็นการลบข้อมูลตำแหน่งท้ายสุดและหน้าสุด
วิธีการสร้าง array ด้วยวิธีต่างๆ
การใช้ ... เพื่อ spread ข้อมูลหรืออ้างถึงข้อมูลก่อนหน้าทั้งหมด และการแยกstring ออกเป็น array
