object เป็น type ชนิดหนึ่งของ data ที่เก็บข้อมูลในรูปของ key เพื่อใช้ในการอ้างถึง และ value คือค่าข้างในโดยที่ จะคล้ายกับ array
ตรงที่ variable คือเหมือนป้ายบอกถึง ซึ่งในการเรียกมันออกมาจะเป็นการอ้างถึงหรือ preference ถ้า array คือกล่องหลายๆกล่อง ที่เรียงต่อกันมีลำดับ object คือกล่องหลายๆกล่องกระจัดกระจายโดยที่มีป้ายแปะบอกชื่อคือ key
และสิ่งที่อยู่ข้างในคือ value โดยที่มันเป็น mutable เหมือนๆห้องหนึ่งที่ใครก้ตามสามารถเข้ามาจัดการข้างในห้องได้ ตกแต่งอย่างไงก็ได้ แต่ต่อให้
เราทำการตกแต่งห้องเหมือนกันเราก็จะบอกว่ามันคือห้องเดียวกัน เหมือนฝาแฝดสองคนที่เหมือนกัน แต่พวกมันก็ไม่ใช่คนเดียวกัน หรือ สินค้าที่ผลิตออกมาเราก็จะไม่บอกว่ามันเป็นสิ่งของที่เหมือนกัน เราเรียกมันว่า
ชิ้นที่1 หรือชิ้นที่2
object สามารถเก็บ value ได้หลาย type สามารถเอา object นำไปซ้อนข้างในได้ หรือเก็บเป็นfunction ก็ได้
เราสามารถเขียน object ได้หลายแบบ ถ้าเป็น class จะมีคุณสมบัติในการส่งต่อ protertype ให้ class อื่นได้ ให้อารมณ์เหมือนพิมพ์เขียว หรือตัวต้นแบบ

แต่วิธีการในการเช็คว่า object ทั้งสองตัวนั้นเหมือนกันไหม จำเป็นจะต้องเช็คตัวของ key และ value หรือเปลี่ยนมันเป็นstring โดยใช้ JSON.stringify และเช็คค่าของ stringify
โดยตัวของ JSON คือการเขียนตัวของข้อความในรูปของ Object เพื่อง่ายต่อการส่งมอบและเชื่อมต่อระหว่าง Frontend และ BackEnd
