Array Method "ArrayIs" เอาไว้สำรหับตรวจเช็คว่า parameter ที่รับเข้ามาเป็น array หรือไม่ ถ้าใช่ return กลับมาเป็น true
concat คือ array method ที่ใช้ในการรวม 2 parameter เข้าด้วยกัน คล้ายกับการใช่ spread
tostring คือการเปลี่ยน array ให้กลายเป็น string join ก็เหมือนกันแต่เราสามารถเปลี่ยน , ที่ปกติจะเป็น default สามารถตั้งเป็นตามที่เราต้องการได้
reverse คือการสลับarray จากหลังสุดมาหน้าสุด และจากหน้าสุดไปหลังสุด
แต่เราสามารถใช้ sort เพื่อทำการเรียงลำดับ array ได้ จากน้อยไปมาก แต่ถ้าเป็นตัวเลขเราจะไม่สามารถเรียงลำดับได้เพราะว่ามันจัดเรียงจาก unicode เราต้องทำการใช้ parameterเป็น
compare function เพื่อเปรียบเทียบค่าข้างในด้วย sort(a,b) => a-b ถ้าผลลัพย์พารามิเตอร์ติดลบ a จะอยู่ตัวแรกสุดและสลับเอา b ไปท้ายสุด ทำจนครบทุกตัว ถ้าเป็น b-a ก็จะเป็นจากมากไปน้อย
toLocaleString() ใน JavaScript คือ method ของ Object เช่น Number, Date หรือ Array ที่ใช้สำหรับแปลงค่าวัตถุนั้นเป็น สตริงในรูปแบบที่เหมาะสมกับท้องถิ่น
js module คือการexport และ import type ต่างๆใน js ไปในไฟล์อื่นๆ เพื่อลดความซับซ้อนของมัน โดยเราสามารถทำ export class object หรือ ฟังก์ชันก็ได้ วิธีการexport มีหลายวิธี ทั้งแบบที่เวลา import
